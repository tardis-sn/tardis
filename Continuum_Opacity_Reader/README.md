This is the Continuum Opacity Source Reader file in the TARDIS Project. This file is 
meant to be able to read data from Scientific Literature in the form of Gzip or PDF files
and return a pandas Dataframe that can be used to feed into the project for further use. 
It is an initial prototype and lacks the polish there should be in the later versions 
and the current goal is to have it be more accurate for the project currently.
