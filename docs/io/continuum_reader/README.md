Contnuum Opacity Source Reader:
Objective: To standardize the importing and reading from Scientific Papers of .gz files. 

Method: Used .gzip module for reading .gz files and pandas module to convert the data in the file 
into a Pandas dataframe. 

Ensured that the outcome is generalized for most files, ensured tight documentation for integration with Sphinx environment
as well as created a Jupyter Notebook with a walkthrough for working with the module. 

For a demonstration, refer to tutorial_run_cor>test.ipynb given alongside for the walkthrough of the module.
